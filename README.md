# Seeing-Clearly

This project implements a real-time facial expression recognition system using a fine-tuned ResNet-18 model trained on the FER-2013 dataset. The system detects faces from a webcam and predicts emotional states to assist users with emotion recognition.

Step 1: Download the path file here
https://drive.google.com/drive/folders/1HVltZlAXNfmxvADoRKGA8KqWAeBF6guA?usp=sharing

Step 2: Run the emotion_camera_demo.pynb

Optional
The code for how the models were trained can be found in "" if you seek to run it from stratch.

This will work directly with your camera app so make sure the device you use has one.
